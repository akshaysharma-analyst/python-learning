# Study Notes App (Python Beginner Project)

## Description
This is a simple Python program that allows a user to write a study note, saves it into a file, and then reads it back and displays it.

## Features
- Takes user input
- Saves data to a text file
- Reads data from a file
- Displays saved content

## What I learned
- Python input and output
- File handling (read/write)
- Using `with open()` in Python
- Basic program structure

## How to run
Run the Python file in any Python environment:

```bash
python study_notes_app.py
